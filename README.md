# WHHS Website

Static multi-page website for Woodland Hills Schools (WHHS).

## Structure
- `index.html` and supporting pages: `about.html`, `academics.html`, `admissions.html`, `contact.html`, `contacts.html`, `dashboard.html`, `facilities.html`, `high-flyers.html`, `news.html`, `policies.html`, `timetable.html`
- Shared assets in project root: `styles.css`, `app.js`, `whhs-logo.webp`

## Run locally
Open `index.html` in your browser.

## Notes
All asset references point to the project root (no `assets/` folder).
